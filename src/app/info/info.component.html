Welcome to my project!

This project utilizes Dijkstra's Algorithm to find the shortest path from the Start node to the End node.

The Start node is represented by a green node as seen on the right, and the End node is represented by a red node. You can easily move these nodes around by clicking and dragging them.

To draw walls, click and drag your mouse on any open node to create a line of black walls. If you want to erase a wall click on the desired wall, you can click and drag to delete multiple walls. These walls are taken into account when calculating the shortest path.

To find the shortest path, simply click the Start button. You can reset the board and generate new Start and End nodes by clicking the reset button, or generate a grid with randomized walls by clicking the randomize button.

You also have the option to change the colors of both the visited nodes and the path drawn.

How does this Work?

This project utilizes Dijkstra's Algorithm to find the shortest path from the Start node to the End node.

The Start node is represented by a green node, and the End node is represented by a red node. You can easily move these nodes around by clicking and dragging them.

To draw walls, click and drag your mouse on any open node to create a line of black walls. If you want to erase a wall click on the desired wall, you can click and drag to delete multiple walls. These walls are taken into account when calculating the shortest path.

To find the shortest path, simply click the Start button. If you want to reset the board and clear all nodes, click the Reset button. To generate a grid with randomized walls, click the Random Layout button. You also have the option to change the colors of both the visited nodes and the path drawn.
